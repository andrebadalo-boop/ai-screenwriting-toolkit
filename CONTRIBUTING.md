# Contributing

Thank you for helping make this toolkit better for writers everywhere. 🙏

## What we're looking for
- **Translations** of prompts/templates (PT, ES, FR especially welcome)
- **New prompts** for stages not yet covered
- **Case studies**: how a prompt was used on a real project (anonymized is fine)
- **Fixes**: typos, unclear wording, broken structure

## How to contribute
1. Fork the repository.
2. Create a branch (`add/character-flaw-prompt`).
3. Make your change, following the existing format (see any file in `prompts/`).
4. Open a Pull Request with a short description of *why* it helps writers.

## Format for a new prompt
Please keep the same sections used across the toolkit:
**Objective → When to use → The prompt → Variables → Example → Notes.**

## Ground rules
- Be kind and constructive.
- Keep attribution (CC BY 4.0).
- No proprietary studio material — only your own or public-domain content.

By contributing, you agree to license your contribution under CC BY 4.0.
